Pipes are used to format data using '|' before showing in html

Without Pipe With Pipe
{{showcase}} uppercase pipe= {{showcase | uppercase}}
{{showcase}} lowercase pipe ={{showcase | lowercase}}
{{showcase}} title pipe ={{showcase | titlecase}}
{{student}} json pipe ={{student | json}}
{{date}} Date pipe ={{date | date:'hh:mm:ss'}}

Note: Json pipe will show Object data as string
